# ☕ Brewed Coffee Shop Website

A modern, responsive coffee shop website with a warm and cozy design. Built to showcase menu, location, and contact information — perfect for small local businesses.

---

## 🔗 Live Demo
Check it live here: [Brewed Coffee Shop](https://brewedcoffeeshop.netlify.app/)

---

## 🚀 Features
- Fully responsive design (mobile, tablet, desktop)
- Hero section with call-to-action
- Menu section with prices and images
- Contact section with form and Google Maps location
- Smooth hover effects on buttons
- Clean, modern UI with warm & cozy colors

---

- HTML
- CSS
- JavaScript
